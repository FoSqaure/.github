## 🎄 **FoSquare - 유저가 원하는 게시판을 직접 생성할 수 있는 게시판 사이트**  

**FoSqaure**의 핵심 기능
- **테이블 생성 기능**으로 사용자의 실시간으로 게시판을 직접 생성할 수 있습니다.  
- **이미지 동영상 업로드**로 유저 간 정보공유를 보다 원활하게 할 수 있습니다.  
- **이메일 인증**을 통해 무분별한 회원가입을 방지할 수 있습니다.
- **SQL Injection 대비**를 통해 보안을 보완 하였습니다.
